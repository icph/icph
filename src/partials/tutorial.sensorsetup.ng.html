Setup a pre-defined sensor in Node-Red

Overview

Connect a sensor to the gateway and use Node-Red to automatically read temperature, edit a sample Node-Red application, and deploy it to read humidity

Steps

Step 1

Connect sensor

Plug the RH-USB temperature / humidity sensor into any open USB port on the gateway.

Step 2

Open Node-Red

Navigate to Sensors then click Program Sensors to open Node-Red.

Step 3

Explore default flow

A default Node-Red flow has been included. In this flow, an inject node triggers the RH-USB sensor to provide the current temperature and humidity every second. The temperature data goes through a function node that converts the temperature reading to Celsius and rounds the value down to the nearest tenth. That value goes through the chart node for formatting and then to the mqtt node. The Intel® IoT Gateway Developer Hub watches the mqtt service on the gateway and automatically displays and publishes the data to the /sensor topic. Note how the Humidity chat node is not connected to the RH-USB node.

Step 4

Show Info panel

An Info panel shows the details of each node. To open the Info panel, click the drop-down menu in the upper right corner of Node-Red, and then choose Sidebar \ Toggle Sidebar. Click each node to see the details.

Step 5

Add Humidity reading to flow

Connect the second output of the RH-USB sub-flow node to the input of the Humidity chart node.

Click the output point (on right side) of RH-USB and drag a connection to the input (left side) of Humidity node.

Step 6

(Optional) Modify the Function node

The Function node allows you to pass each message though a JavaScript function. Double click teh "F to C" Function node to open the edit dialog.

Step 7

Deploy the updated flow

The Deploy button is found in the upper right corner of Node-red. Click the Deploy button.

Step 8

Test sensor

View the Intel® IoT Gateway Developer Hub Dashboard to see the Humidity gauge that is now included and displays humidity data.

Test the sensor by gently breathing on it. You should see the temperature and humidity values change immediately.

Step 9

(Optional) Export/Import a flow

An existing flow in JSON format can be imported into Node-Red. To do this, click the drop-down menu in the upper right corner of Node-Red, and then choose Import \ Clipboard. Copy the flow JSON into the Clipboard.

The current flow on Node-Red can be exported to JSON format. Select all the flow components first and then follow the steps described above but choose Export instead of Import.

Step 10

(Optional) Create a new node

A new node can be created if the existing nodes do not meet your needs. Please follow the official Node-Red documentation.
